# Web Daily Lab

This is a very easy project for beginners.

You can use this repository to:
- Write daily notes
- Try small ideas
- Practice editing files
- Build a habit of GitHub commits

How to use:
1. Open any file
2. Add a small change or note
3. Save the file
4. Commit your update

Goal:
Practice something small every day.

Notes:
Write your daily lab notes below.
